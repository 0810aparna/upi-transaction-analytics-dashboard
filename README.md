# UPI Transaction Analytics Dashboard

## Overview
This project analyzes UPI transaction trends using Python and Power BI.

## Features
- Transaction analysis
- State-wise trends
- Merchant category analysis
- Visualization dashboards

## Tech Stack
- Python
- Pandas
- Matplotlib
- SQL
- Power BI

## Dataset
UPI Transactions 2024 Dataset

## How to Run

```bash
pip install -r requirements.txt
python main.py
